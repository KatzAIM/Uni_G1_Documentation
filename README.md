# Unitree G1 Robot SDK Documentation

This repository contains the detailed documentation for the Unitree G1 Robot Python SDK.

## Access the Documentation Site

The full documentation, formatted as a web page with navigable tabs, can be accessed via GitHub Pages:
[https://katzaim.github.io/Uni_G1_Documentation/](https://katzaim.github.io/Uni_G1_Documentation/)

**Note**: If the link above does not work, please ensure that GitHub Pages is enabled for the `main` branch in your repository settings. You can find this under `Settings` -> `Pages` in your GitHub repository.

## Overview of Documentation

The documentation provides comprehensive details on using the Unitree G1 Robot Python SDK, covering:

*   **Overview**: General introduction, SDK version, GitHub repository link, and installation instructions.
*   **Run Instructions**: Detailed steps for setting up the network connection with the robot and executing example code.
*   **Arm SDK**: API reference for controlling the robot's arm, including available actions and methods.
*   **Audio SDK**: API reference for text-to-speech, LED control, and audio playback features.
*   **Locomotion SDK**: API reference for controlling the robot's movement, posture, and various locomotion actions.
*   **Examples**: Practical code snippets demonstrating how to use the SDK's functionalities.
